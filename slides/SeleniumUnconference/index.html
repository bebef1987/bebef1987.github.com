What do we do?

  • Ensure high quality of Mozilla web applications
  • We test aprox 15 sites daily
  • Run manual tests using Moztrap for all the sites
    • https://moztrap.mozilla.org/
  • Write automated tests for all the sites
    • We use Selenium and Python
  • Security testing
  • Performance testing

Style Guide

        # Good
        def click_login(self)
          self.selenium.find_element(*self._login_locator).click()
        
        # Bad
        def click_login(self)
          if not self.is_user_logged_in:
            self.selenium.find_element(*self._login_locator).click()
          else:
            pass
